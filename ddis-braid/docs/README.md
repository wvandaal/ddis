## Directory Structure

- **`HARVEST.md`** — Session log. Read at start, write at end.
- **`guide/`** — Per-namespace implementation guides. See `guide/README.md` for build order.
- **`design/`** — Design decision index (ADRS), failure mode catalog, defect tracking.
- **`audits/`** — Gap analysis, formal audit reports, research studies.
- **`history/`** — Transcripts, references, explorations (archival). See `history/README.md`.
